WSO2 Enterprise Integrator Monitor

    WSO2 Enterprise Integrator Monitor is a tool that can be used to monitor an instance of Enterprise Integrator(EI)
    and find the reason for an issue occur in the EI. When an issue occurs in a production server, to investigate
    the incident, we need to fetch several information from the server instance. This tool automatically extracts useful
    information from the server when there are potential issues.

The Monitored details regarding the last issue is as follows.

Heap Memory

CPU Memory

LoadAverage

ThreadStatus
